auth

(
  • options
  • [cb]
)
Promise async

Defined in lib/lp-sdk.js:70

Authenticate a developer, and provide a token for La Poste Open API.

The resulting token is stored as "accessToken" instance attribute.

Parameters:

  • options Object

    Options object.

    • consumerKey String

      The consumer key.

    • consumerSecret String

      The consumer secret.

    • username String

      The developer account username.

    • password String

      The developer account password.

  • [cb] Function optional

    The callback(err, result) function to be called when the request is fulfilled, if not defined a promise is returned.

Returns:

Promise:

A promise, fulfilled when the request is done.

Throws:

HttpError:

An HTTP error including status code and body.

Example:

 var lpSdk = require('lp-sdk')
   , lp = new lpSdk.LaPoste();

 lp.auth({
   consumerKey: 'zzzz-aaa-ccc',
   consumerSecret: 'aaaa-ccc-rrr',
   username: 'john',
   password: 'mysecret'
 }, function (err, result) {
   if (err) {
     console.error(err);
     return;
   }
   console.log('result :', result);
 });
